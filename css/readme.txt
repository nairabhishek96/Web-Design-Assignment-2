Abhishek's Personal Professional Portfolio

1) My Personal Professional Portfolio has a two column layout. It has total 7 sections.
2) The header consists of the introduction and the image. When the user clicks on the Home button of the nav bar, the introduction and link to various social media tabs appear in the form of clickable images.
3) When the user clicks on the about button the user is redirected to the about me section where the description is given along with the technical skills.
4) In the education section the details about education are listed. 
5) In the Certifications section an image gallery is used to display the Certifications.
6) The Project sections lists down all my personal projects with a detailed explanation of each of the project.
7) In the contact me form the basic info like first name, last name, email, project and any additional message can be mentioned and the details are mailed to provided email address.
